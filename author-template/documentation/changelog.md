# Changelog

All notable changes to the Aria Vance Author Template will be documented in this file.

## [1.1.0] - 2026-04-06
### Added
- Complete **SEO** metadata (Title, Description, Keywords) for all pages.
- **OG (Open Graph)** and **Twitter Card** social sharing tags.
- **JSON-LD** schema for Author/Person and CreativeWorks.
- `robots.txt` and `sitemap.xml` for search crawler optimization.
- Detailed **Documentation** sub-folder with 6 new README files.

### Changed
- All images migrated to **WebP** for better LCP (Largest Contentful Paint).
- All Google Fonts moved from CDN to **local hosting** in `assets/fonts/`.
- Images organized into `hero/`, `ui/`, and `content/` folders.
- Enhanced **Hero Image** with new premium cinematic star-born art.

### Fixed
- Navigation link paths to ensure consistency in the nested `pages/` structure.
- Favicon references and fallback styles for image-loading states.

## [1.0.0] - Initial Release
- Core HTML structure and layout.
- Dark mode Toggle / RTL support.
- GSAP and Swiper integrations.
- Basic placeholder sections.
