## 2. Common Issues (FAQ)
- **RTL Not Working?**: Ensure the `assets/css/rtl.css` file is present and not modified to break its flexbox logic.
- **Dark Mode Not Sticky?**: Check the `localStorage` key in `main.js`. It should be "theme".
- **Images Are Missing?**: Verify the extensions. We use `.webp`. If you've renamed them, ensure the HTML reflects those changes accurately.
